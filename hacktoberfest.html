Hacktoberfest 2022

Steps to make PR for Hacktoberfest

  1. Create a GitHub account, if you don't already have one.
  2. Register for Hacktoberfest by going on Hacktoberfest website/li>
  3. First open the project on the github that you want to contribute.

    There must be label of Hacktoberfest

  4. Fork the project
  5. And you can see url : yourusername/projectname
  6. And click on add file and create new file
  7. and make your file name like : test.py and click enter and one file is created.
  8. and change or add some programs on it
  9. and write something in describtion and commit it
  10. and click on open pull request

Like these ways, you can create your first pr

If you are confused about something, google it or you can message me
